# Курсовой проект

Заданием было генеративное меню, а вышло мини-приложение для автоматизации составления рациона питания с учетом заданных норм калорийности.

## Особенности проекта

В рамках реализации **Варианта №5 (Составление меню по заданным калориям)** проект был расширен дополнительным функционалом для повышения реалистичности и практической пользы:

- **Smart Generator**: Алгоритм автоматического подбора продуктов под остаток дневного лимита калорий.
- **Трекер калорий**: Визуальный мониторинг потребления КБЖУ в реальном времени с использованием динамических прогресс-баров.
- **Управление продуктами**: Полноценный справочник с возможностью добавления, редактирования и удаления продуктов.
- **Архив (История)**: Возможность просмотра детальных отчетов по рационам за прошлые дни.
- **Адаптивный дизайн**: Интерфейс оптимизирован для работы как на десктопах, так и на мобильных устройствах.

## Стек технологий

- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript
- **Database & ORM**: SQLite + Prisma ORM
- **Styling**: Tailwind CSS + Shadcn UI
- **Form Management**: Zod + React Hook Form
- **Icons**: Lucide React

## Быстрый старт

### Требования

Для запуска проекта нужна **Node.js** (рекомендуемая версия 20+). Установить её можно с оф. сайта, через `scoop`, либо через менеджеры версий (`NVM`, `Volta`).

### Установка зависимостей

В корне проекта выполните команду:

```bash
npm install
```

И после этого

```bash
npm run dev
```
